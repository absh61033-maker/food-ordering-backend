# Food Ordering - Backend ⚙️

The backend API for the Food Ordering application, built with Node.js and Express.

## ✨ Features

- User authentication (JWT)
- Order management
- Menu/food item management
- Email notifications

## 🛠️ Tech Stack

- Node.js
- Express.js
- MongoDB
